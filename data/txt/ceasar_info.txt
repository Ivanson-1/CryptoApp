Шифр Цезаря — это простой метод шифрования, который сдвигает каждую букву текста на фиксированное число позиций в алфавите. 

1. **Шифрование**:
   - Для каждой буквы исходного текста вычисляется новая буква по формуле:  
     C = (P + K) mod 26,  
     где P — индекс исходной буквы, K — ключ (сдвиг), C — индекс шифротекста.

2. **Расшифровка**:
   - Для каждой буквы шифротекста вычисляется исходная буква по формуле:  
     P = (C - K) mod 26.

3. **Пример**:
   - При ключе ( K = 3 ): A → D, B → E, C → F, и так далее.

Шифр Цезаря легко взломать с помощью перебора всех возможных ключей (всего 25 вариантов).
